🔧 One-time setup needed

The Gaspy API requires a small cloud proxy. Set one up free in 2 minutes using Cloudflare Workers — no Node.js, no server required.

1
Create a free Cloudflare account Sign up at cloudflare.com (free tier is plenty).
2
Create a Worker and paste the code Go to Workers & Pages → Create → Hello World, then replace all the code with the contents of worker.js and click Deploy.
3
Paste your Worker URL below It looks like https://your-worker.workers.dev
